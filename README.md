# JSBattle();

A simple Vue 3 webpage with some challenges you can do. It will show your output and it is made with levels in mind, for example;
Level 0: make a multiplay function,
Level 1: Make a OR function (||).

## Build
Do 
`yarn` and
`yarn build`.
